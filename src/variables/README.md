#  

## 常量与不可变变量的区别

定义方式
| 特性 | 常量 (`const`) | 不可变变量 (`let`)     |
| -------- | ------------ | ----------------- |
| 关键字 | `const`      | `let`（默认不可变） |
| 是否需要类型注解 | ✅ 必须显式写出类型 | ❌ 可以类型推导 |
| 是否可变 | ❌ 永远不可变 | ✅ 可以使用 `mut` 变为可变 |

生命周期与作用域
| 特性 | 常量 (`const`)     | 不可变变量 (`let`)    |
| ---- | ---------------- | ---------------- |
| 生命周期 | 编译期确定，**无运行时开销** | 运行时创建，生命周期受作用域限制 |
| 可见范围 | 可以在全局作用域定义 | 只能在函数或块作用域中定义 |

用途与语义
| 特性 | 常量 (`const`)    | 不可变变量 (`let`)    |
| ---- | --------------- | ---------------- |
| 使用场景 | 常用于**全局常量**或配置值 | 通常用于函数内部的数据绑定 |
| 存储方式 | 嵌入到程序的二进制中 | 在运行时栈上分配内存（或被优化） |
