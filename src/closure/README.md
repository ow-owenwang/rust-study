#  

Rust 中的\*\*闭包（closure）\*\*是一种能够捕获其环境的匿名函数。闭包是函数式编程中的重要概念，Rust 对它的支持既强大又安全。下面从几个方面介绍
Rust 闭包的特点、应用场景，并给出代码示例。


## 闭包的特点

1. **可以捕获环境中的变量**

    * 闭包会自动推断需要捕获的变量及其方式（按借用、可变借用、值移动）。
2. **类型推断能力强**

    * 编译器通常可以推断闭包的参数和返回值类型，不需要显式标注。
3. **可作为函数参数或返回值**

    * 闭包可以作为参数传递给函数，也可以作为函数的返回值。
4. **三种捕获方式**

    * `Fn`：通过不可变借用捕获变量。
    * `FnMut`：通过可变借用捕获变量。
    * `FnOnce`：通过值移动捕获变量，只能调用一次。

## 闭包在实际中的应用场景

1. **作为回调函数使用**

    * 如迭代器中的 `map`, `filter` 等函数。
2. **自定义控制流程**

    * 如模拟延迟执行、条件执行。
3. **函数工厂**

    * 闭包作为返回值，动态创建不同行为的函数。
4. **并发编程**

    * 在线程中传递闭包（需要 `Send + 'static` trait）。
